
Explicación de los archivos:

- creación_dataset_train: Script de Python correspondiente al proceso llevado a cabo para la creación del dataset de entrenamiento.

- creación_dataset_test: Script de Python correspondiente al proceso llevado a cabo para la creación del dataset de test.

- RRNN_FilmFamePredictor: Script de Python correspondiente a todo el proceso, desde el principio al final, llevado a cabo para la construcción de la red neuronal.

- App_FilmFamePredictor: Script de Python correspondiente a la red neuronal integrada en la página web y funcionando.

- sequence_model.sav: El modelo, la red neuronal.
